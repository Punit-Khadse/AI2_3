Insurance Cost Prediction Project

📌 Overview

This project predicts medical insurance charges using machine learning.

📊 Dataset

The dataset contains:

- Age
- BMI
- Smoking status
- Charges (target)

⚙️ Steps Performed

- Data loading
- Data preprocessing
- Train-test split
- Model building

🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn

▶️ How to Run

1. Open notebook in Jupyter
2. Run all cells
